基本概念

1.2什么是算法？

1.算法

  • 一个有限指令集
  • 接受一些输入（有些情况下不需要输入）
  • 产生输出
  • 一定在有限步骤后终止
  • 每一条指令必须
    • 有充分明确的目标，不可以有歧义
    • 计算机能处理的范围之内
    • 描述应该抽象，不依赖于任何一种计算机语言以及具体实现手段

例1.选择排序算法的伪码描述

void SelectionSort(int List[], int N)
{
    /*将N个整数List[0]...List[N - 1] 进行从小到大排序*/
    for(i = 0; i < N; i++){
        MinPosition = ScanForMin(List, i, N - 1);
        /*从List[i]到List[N - 1]中找最小元，并将其位置赋给MinPosition*/
        Swap(List[i], List[MinPosition]);
        /*将未排序部分的最小元换到有序部分的最后位置*/
    }
}

抽象的体现：

- List是数组还是链表？
- Swap用函数还是用宏去实现？

2.什么是好的算法？

  • 空间复杂度S(n) —— 根据算法写成的程序在执行时 占用存储单元的长度。这个长度往往与输入数据的规模有关。空间复杂度过高的算法可能导致使用的内存超限，造成程序非正常中断。
  • 时间复杂度T(n) —— 根据算法写成的程序在执行时耗费时间的长度。这个长度往往也与输入数据的规模有关。时间负责度过高的低效算法可能导致我们在有生之年都等不到运行结果。

1.1 例2 递归打印出1-N

void PrintN(int N)
{
    if(N){
        PrintN(N - 1);
        printf("%d\n", N);
    }
    return;
}

S(N)= CN,空间复杂度是一个N的函数，C是一个常亮，空间复杂度随着N的增加而变大。
每个程序可用空间有限，当N很大，需要的空间大于当前程序拥有的空间时，程序异常退出。不能输出正确结果。

1.1 例3 多项式求值

double f(int n, double a[], double x)
{
    int i;
    double p = a[0];
    for(i = 1; i <= n; i++){
        p += (a(i) * pow(x, i));
    }
    return p;
}

执行了多少次乘法？
for循环执行了n次，
pow(x,i)是求x的i次方，执行了i-1次乘法，
每一次循环中，执行了i次乘法。
一共执行了(1+2+…+n) = (n ^2 + n) / 2 次乘法。
时间复杂度 T(n) = C1 n^2 + C2 n

double f(int n, double a[], double x)
{
    int i;
    double p = a[n];
    for(i = n; i > 0; i--){
        p = a[i - 1] + x * p;
    }
    return p;
}

执行了多少次乘法？
每次循环执行1次乘法，执行了n次循环，一共执行了n次乘法。
时间复杂度 T(n) = C n

什么是最好的算法？

  • 在分析一般算法的效率时，我们经常关注下边两种复杂度
    • 最坏情况复杂度 T worst (n)
    • 平均复杂度 T avg (n)

数据结构1.1 基本概念——什么是数据结构？

基本概念

1.1什么是数据结构？

1.例1.如何在书架上摆放图书？

图书摆放要求2个操作方便实现：

  • 操作1：新书怎么插入？
  • 操作2：怎么找到指定的书？

方法1：随便放
插入容易，查找难
方法2：按书名的拼音字母顺序排放
二分法
如何插入？
旧书不断移动，插入难
方法3：分类
书架分几个区域，每个区域摆放一类书，每类书按字母顺序排放
如何查找？
先找类别，再二分查找
如何插入？
先确定类别，再根据字母顺序插入。

  • 每类分多少书架？类别分多细？ *

解决问题方法的效率，跟数据的组织方式有关

2.例2：写一个函数printN，输入正整数N，顺序打印出1-N的全部正整数。

方法1：循环实现

void printN(int N)
{
    int i;
    for(int i = 1; i <= N; i++){
        printf("%d\n", i );
    }
    return;
}

方法2:递归实现

void printN(inr N)
{
    if(N){
        printN(N - 1);
        printf("%d\n", N);
    }
    return;
}

当 N = 10w 时，递归不能正常打印。

解决问题方法的效率，跟空间的利用效率有关

3.例3.计算给定多项式在给定点x处的值

计算两种方式耗时：
clock() : 捕捉从程序开始运行到clock()被调用时所耗费的时间，单位是clock tick，“时钟打点”
CLK_TCK : 机器时钟每秒所走的时钟打点数

#include <stdio.h>
#include <time.h>
clock_t start, stop;
double duration;

int main()
{
    start = clock();
    MyFunction(); /*测试函数*/
    stop = clock();
    duration = ((double) (stop - start)) / CLK_TCK;

    return 0;
}

运行一次，duration可能为0.
解决方式：重复运行多次，再取平均值。

解决问题方法的效率，跟算法的巧妙程度有关

总结

  • 数据对象在计算机中的组织方式：
    • 逻辑结构
    • 物理存储结构
  • 数据对象必定与一系列加在其上的操作相关联
  • 完成这些操作所用的方法就是算法

4. 抽象数据类型

  • 数据类型
    • 数据对象集
    • 数据集合相关联的操作集
  • 抽象： 描述数据类型的方法不依赖于具体实现
    • 与存放数据的机器无关
    • 与数据存储的物理结构无关
    • 与实现操作的算法和编程语言均无关
      即 只描述数据对象集和相关操作集“是什么”，不涉及“如何做到”的问题

例4：“矩阵”的抽象数据类型定义

  • 类型名称：矩阵
  • 数据对象集：一个MxN的矩阵

由MxN个三元组构成，其中a是矩阵元素的值，i是元素所在的行号，j是元素所在的列号。（a是抽象的，不管a具体是什么，二维数组？一维数据？十字链表？）

  • 操作集：对于任意矩阵 A、B、C 属于 Matrix，以及整数 i、j、M、N
    • Matrix Create(int M, int N) 返回一个M x N的空矩阵
    • int GetMaxRow(Matrix A) 返回矩阵A的总行数
    • int GetMaxCol(Matrix A) 返回矩阵A的总列数
    • ElementType GetEntry(Matrix A, int i, int j) 返回矩阵A的第i行、第j列的元素
    • Matrix Add(Matrix A, Matrix B) 如果A和B的行、列数一致，则返回矩阵C = A + B，否则返回错误标志。（C = A + B是抽象的，先按行加？先按列加？什么语言？）
    • Matrix Multiply(Matrix A, Matrix B) 如果A的列数等于B的行数，则返回矩阵C = A B，否则返回错误标志。
