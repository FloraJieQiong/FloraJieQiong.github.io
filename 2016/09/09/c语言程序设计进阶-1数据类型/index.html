c语言程序设计进阶-1数据类型

1.0 数据类型

sizeof
是一个运算符，给出某个类型或变量在内存中 所占据的字节数

  • 是静态运算符，它的结果在编译时刻就决定了

  • sizeof括号里的运算是不会做的

  • sizeof(int)

  • sizeof(i)

    #include <stdio.h>
int main()
{
    int a;
    a = 6;
    printf("sizeof(a) =%ld\n", sizeof(a));//4
    printf("sizeof(int)=%ld\n", sizeof(int));//4
    printf("sizeof(a++)=%ld\n", sizeof(a++));//4 运行完 a=6
    printf("sizeof(a+1.0)=%ld\n", sizeof(a+1.0));//8 因为1.0是double，double的字节数是8

}

1.1 整数类型

1.1.1 整数类型

#include<stdio.h>

int main()
{
    printf("sizeof(char) = %d\n", sizeof(char));
    printf("sizeof(short) = %d\n", sizeof(short));
    printf("sizeof(int) = %d\n", sizeof(int));
    printf("sizeof(long) = %d\n", sizeof(long));
    printf("sizeof(long long) = %d\n", sizeof(long long));
}

结果：

sizeof(char) = 1
sizeof(short) = 2
sizeof(int) = 4
sizeof(long) = 4
sizeof(long long) = 8

整数

  • char ：1字节（8bit）
  • short：2字节
  • int：取决于编译器（CPU），通常的意义是“1个字”
  • long：取决于编译器（CPU），通常的意义是“1个字”
  • long long： 8字节

1.1.2 整数的内部表达

二进制负数

  • 1个字节可以表达的数：
    • 00000000 - 11111111（0-255）

  • 如何表达负数？

    • 仿照十进制，有一个特殊标志表示负数
    • 取中间的数为0，如10000000表示0，比它小的是负数，比它大的是正数
    • 补码

      补码

      -1如何表示，我们希望 -1 + 1 –> 0
      0 –> 00000000
      1 –> 00000001
      11111111 + 00000001 –> 100000000
      因为0-1 –> -1,所以 -1 =
      (1)00000000 - 00000001 –> 11111111

  • 11111111被当做纯二进制看待时，是255；被当做补码看待时是-1

  • 同理，对于-a，补码是 0-a,实际是 2^n -a，n是这种类型的位数
  • 补码的意义就是 拿补码和原码 可以加出一个溢出的“零”

1.1.3 整数的范围

数的范围

  • 对于一个字节（8位），可以表达的是：
  • 0000000 - 11111111
  • 其中：
    • 00000000 –> 0
    • 11111111 ~ 10000000 –> -1 ~ -128
    • 00000001 ~ 01111111 –> 1 ~ 127

demo:

#include<stdio.h>
int main()
{
    char c = 255;
    int i = 255;
    printf("c=%d, i=%d", c, i);//c=-1, i=255
    //c 11111111
    //i 00000000 00000000 00000000 11111111

    return 0;
}

整数的范围

  • char： 1字节： -128 ~ 127
  • short : 2字节 : -32768 ~ 32767
  • int ： 取决于编译器（CPU），通常的意义是“1个字”
  • long：4字节
  • long long： 8字节

unsigned

  • 如果一个数字要表达自己是unsigned，可以在后边加 u 或 U，如255U
  • 用l或L表示long（long）

demo:

#include<stdio.h>

int main()
{
    unsigned char c = 255;
    printf("c=%d", c);//c=255
    //c 11111111

    return 0;
}

char : 00000000 - 11111111 表示0-255， -128 ~ 127

整数越界

  • 整数是以纯二进制方式进行计算的，所以：
    • -1 + 1 –> 11111111 + 1 –> 100000000 –> 0
    • 127 + 1 –> 01111111 + 1 –> 10000000 –> -128
    • -128 -1 –> 10000000 - 1 –> 01111111 –> 127

负数 0 正数 构成一个圆
0 减 1 得-1
-1 加 1 得0
127 加 1 得 -128
-128 减1 得 127

demo:获取int最大值

#include<stdio.h>

int main()
{
    int a = 0, b = 0;
    while(++a > 0);
    printf("int 最大值是:%d\n", a-1);

    b++;
    while((a=a/10) != 0)
    {
        b++;
    }
    printf("int最大值位数是%d",b);
    //int 最大值是:2147483647
    //int最大值位数是10
    return 0;
}

1.1.4 整数的格式化

整数的输入输出

  • 只有两种形式：int 或long long
  • %d : int
  • %u : unsigned
  • %ld : long long
  • %lu : unsigned long long

进制

  • 八进制：0开头 ，输出用%o
  • 十六进制: 0x开头，输出用%x,%X。（%x输出10为a，%X输出10为A）
    八进制与十六进制 只是如何把数字 表达为 字符串，与内部如何存储无关

1.1.5 选择整数类型

  • 没有特殊需要，选择int
  • unsigned 只是输出不同，内部计算是一样的

1.2 浮点类型

1.2.1 浮点类型

1.类型：

  • float：

    • 字长 32
    • 范围 ±(1.2010^-38 ~ 3.4010^38),0,±inf,nan
    • 有效数字位数 7

  • double：

    • 字长 64
    • 范围 ±(2.210^-308 ~ 1.7910^308),0,±inf,nan
    • 有效数字位数 15

2.浮点数的输入输出

  • float：
    • scanf : %f
    • printf : %f, %e
  • double:
    • scanf : %lf
    • printf : %f,%e
#include<stdio.h>

int main()
{
    double d = 1234.56789;
    printf("%e,%f\n", d, d);//1.234568e+003,1234.567890
    printf("%E\n", d);//1.234568E+003
    double d1 = 1E-10;
    printf("%E,%.16f\n", d1, d1);//1.000000E-010,0.0000000001000000

    return 0;
}

输出精度

- %和f之间加上 .n 指定输出 小数点后n位，这种输出是 四舍五入的

#include<stdio.h>

int main()
{
    printf("%.3f\n", -0.0049);//-0.005
    printf("%.30f\n", -0.0049);//-0.004899999999999999800000000000
    printf("%.3f\n", -0.00049);//-0.000
    return 0;
}

1.2.2 浮点类型的范围与精度

1.超过范围的浮点数

  • printf输出 ±inf，表示超过范围的浮点数：±∞
  • printf输出 nan，表示不存在的浮点数

–

#include<stdio.h>

int main()
{
    printf("%f\n", 12.0/0.0);// 1.#INF00 视频中输出的是inf
    printf("%f\n", -12.0/0.0);// -1.#INF00  视频中输出的是-inf
    printf("%f\n", 0.0/0.0);// -1.#IND00  视频中输出的是nan
    printf("%d\n", 12/0);// warning: division by zero

    return 0;
}

2.double存储的数值

#include<stdio.h>

int main()
{
    float a, b, c;

    a = 1.345f;
    b = 1.123f;
    c = a + b;

    if(c == 2.468){
        printf("相等\n");
    } else {
        printf("不相等, c= %.10f,或%f\n", c, c);//不相等, c= 2.4679999352,或2.468000
    }

    return 0;
}

默认有小数点的变量类型是 double，不是float
float要在数字后加 f 或 F 表明身份

整数是以 二进制的形式 存在于 计算机内部的，浮点数不是。
浮点数是编码的数字，计算时，不能在 计算机内部 直接以 二进制的方式进行，通常需要由专门的硬件支持。
浮点数的计算机内部表达形式 造成了 不同的现象。

浮点数计算时由专用的硬件部件实现，double与float用的部件不同。
2.468 double 和 float，内部存储不一样

3.选择浮点类型

  • 没有特殊需要，用double
