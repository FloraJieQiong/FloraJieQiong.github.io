2.2.1

什么是堆栈？

算术表达式

  • 组成
    • 运算数：1,2,3
    • 运算符号：+，-，*，/

后缀表达式

  • 中缀表达式：运算符号位于两个运算数之前。a+b*c-d/e
  • 后缀表达式：运算符号位于两个运算数之后。abc*+de/-

计算：62/3-42*+=?
后缀表达式求值策略：从左往右扫描，逐个处理运算数和运算符号。
1.遇到运算数先记住
2.遇到运算符，将与运算符挨着的2个运算数计算
答案是8，T(n)=O(n)

用什么数据结构存储后缀表达式？
需要一种存储方法，能顺序存储运算数，并在需要时倒序输出。
堆栈

堆栈的抽象数据类型描述

堆栈：具有一定操作约束的线性表
只在一端（栈顶，Top）做插入删除

  • 入栈：插入数据
  • 出栈：删除数据
  • 顺序：后入先出（Last In First Out ，LIFO）

2.2.2 栈的顺序存储实现

通常由一个一维数组和一个记录栈顶元素位置的变量组成

#define MaxSize<储存数据元素的最大个数>
typedef struct{
    ElementType Data[MaxSize];
    int Top;
}Stack;

(1)入栈

void Push(Stack *PtrS, ElementType item){
    if(Ptrs -> Top == MaxSize - 1){
        printf("堆栈满");
        return;
    } else {
        PtrS -> Data[++(PtrS -> Top)] = item;
        return;
    }
}

(2)出栈

ElementType Pop(Stack *PtrS){
    if(PtrS -> Top == -1){
        printf("堆栈空");
        return ERROR;//ERROR是ElementType的特殊值，标志错误
    } else {
        return (PtrS -> Data[(PtrS -> Top)--]);
    }
}

[例]请用一个数组实现两个堆栈，要求最大地利用数组空间，使数组只要有空间入栈操作就可以成功。

[分析]两个栈 分别从 数组的两头开始 向中间生长，当两个栈的 栈顶指针相遇时，两个栈都满了。

#define MaxSize<存储数据元素的最大个数>
struct DStack{
    ElementType Data[MaxSize];
    int Top1;//堆栈1的栈顶指针
    int Top2;//堆栈2的栈顶指针    
}S;
//堆栈空
S.Top1 = -1;
S.Top2 = MaxSize;

(1)入栈

void Push(struce DStack *PtrS, ElementType item, int Tag){
    //Tag作为区分两个堆栈的标志，取值为1,和2
    if(PtrS -> Top2 - PtrS -> Top1 == 1){
        printf("堆栈满");
        return;            
    }
    if(Tag == 1){
        //对 堆栈1操作
        PtrS -> Data[++(PtrS -> Top1)] = item;
    } else {
        //对 堆栈2操作
        PtrS -> Data[--(PtrS -> Top2)] = item;
    }
}

(2)出栈

ElementType Pop(struct DStack *PtrS, int Tag){
    //Tag作为区分两个堆栈的标志，取值为1,和2
    if(Tag == 1){
        //对 堆栈1操作
        if(PtrS -> Top1 == -1){
            printf("堆栈1空");
            return NULL;
        } else {
            return PtrS -> Data[(PtrS -> Top1)--];
        }
    } else {
        //对 堆栈2操作
        if(PtrS -> Top2 == MaxSize){
            printf("堆栈2空");
            return NULL;
        } else {
            return PtrS -> Data[(PtrS -> Top2)++];
        }
    }
}

2.2.3 堆栈的链式存储实现

栈的链式存储结构 是 一个单向链表，叫做 链栈。
插入和删除操作 只能在 链栈 的栈顶进行。

typedef struct Node{
    ElementType Data;
    struct Node *Next;
}LinkStack;
LinkStack *Top;

(1)堆栈初始化（建立空栈）

LinkStack *createStack(){
    //构建 一个堆栈 的头结点，返回指针
    LinkStack *S;
    S = (LinkStack *)malloc(sizeof(struct Node));
    S -> Next = NULL;
    return S;
}

(2)判断 堆栈S 是否为空

int IsEmpty(LinkStack *S){
    //空 返回1，非空 返回0
    return (S -> Next == NULL);
}

(3)入栈

void Push(ElementType item, LinkStack *S){
    //将item 压入 堆栈S
    struct Node *TmpCell;
    TmpCell = maoolc(sizeof(struct Node));
    TmpCell -> Element = item;
    TmpCell -> Next = S -> Next;
    S -> Next = TmpCell;
}

(4)出栈

ElementType Pop(LinkStack *S){
    //删除栈顶元素 并返回S的栈顶元素
    struct Node *FirstCell;
    ElementType TopElem;
    if(IsEmpty(S)){
        printf("堆栈空");
        return NULL;
    } else {
        FirstCell = S -> Next;
        S -> Next = FirstCell -> Next;
        TopElem = FirstCell -> Element;
        free(FirstCell);
        return TopElem;
    }
}

2.2.4 堆栈应用：表达式求值

  • 回忆 堆栈 实现 后缀表达式 求值：
    从左到右 读入 后缀表达式 的各项（运算数 或 运算符）
    • 1.运算数：入栈
    • 2.运算符：从 堆栈中 弹出 适当数量 的运算数，计算并结果入栈
    • 3.最后，栈顶元素 就是表达式的结果值

中缀表达式求值

基本策略 ：将中缀表达式 转换为 后缀表达式，求值
[例子]: 2+9/3-5 后缀表达式为：293/+5-

  • 1.运算数相对顺序不变
  • 2.运算符号顺序改变
    • 需要存储“等待中”的运算符号
    • 将当前运算符号 与 “等待中” 的 后一个运算符号 比较

[例]a(b+c)/d 后缀表达式： abc+d/

中缀表达式 如何 转换为 后缀表达式
从头到尾读取 中缀表达式的每个对象，不同对象做不同处理：

  • 1.运算数：直接输出；
  • 2.左括号：压入堆栈
  • 3.右括号：将栈顶的运算符弹出并输出，直到遇到左括号（左括号直接出栈，不输出）
  • 4.运算符
    • 优先级 大于 栈顶运算符，则压入栈；
    • 优先级 小于等于 栈顶运算符，将 栈顶运算符 弹出并输出；再比较 新的栈顶运算符，直到该运算符 优先级 大于 栈顶运算符 为止，然后将 该运算符压入栈；
  • 5.若各对象处理完毕，把堆栈中 存留的 运算符 一并输出

堆栈的其他应用

  • 函数调用 及 递归 实现
  • 深度优先搜索
  • 回溯算法
  • ……
